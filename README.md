# Conformidade DCG - SaaS de Monitoramento Regulatorio

Plataforma SaaS da DCG Corretora de Seguros para monitoramento de obrigacoes legais de empresas brasileiras (DTE, DJE, e-BEF, ECF, PGR, LGPD).

## Stack
- Backend: Node.js + Express + PostgreSQL
- Frontend: HTML/CSS/JS vanilla
- Pagamentos: Mercado Pago
- E-mail: Gmail Workspace (SMTP)
- WhatsApp: Zapi
- Infra: VPS Hostinger + PM2 + Nginx

## Modelo Freemium
- Free: 1 CNPJ - R$ 0
- Starter: 3 CNPJs - R$ 19,90/mes
- Pro: 10 CNPJs - R$ 59,90/mes
- Agency: 50 CNPJs - R$ 149,90/mes
