3. Dados Coletados

O tratamento de dados varia conforme o tipo de interacao com a plataforma:

CAMADA A — VISITANTE DA AREA PUBLICA

Ao navegar pelo site sem criar conta, coletamos apenas:

  • Dados de navegacao (endereco IP, tipo de navegador, paginas acessadas)
  • Cookies estritamente necessarios para funcionamento tecnico

Quando o visitante realiza uma consulta publica de CNPJ, o sistema processa dados cadastrais publicos disponibilizados pela Receita Federal. Esses dados nao sao armazenados de forma permanente vinculada ao visitante.

CAMADA B — USUARIO DA AREA LOGADA

Ao criar uma conta, coletamos e armazenamos:

Dados cadastrais

  • Nome completo
  • Endereco de e-mail
  • Numero de telefone/WhatsApp
  • Senha (armazenada com criptografia bcrypt, nunca em texto puro)

Dados de uso do servico

  • CNPJs adicionados para monitoramento
  • Preferencias de alerta (e-mail, WhatsApp)
  • Historico de verificacoes e status das obrigacoes
  • Registros de acesso (data, hora, IP) para seguranca

5. Compartilhamento de Dados

Seus dados podem ser compartilhados exclusivamente com:

  • Provedores de infraestrutura: hospedagem (Hostinger), e-mail (Google Workspace), mensageria (WhatsApp Business API/Zapi)
  • Processadores de pagamento: Mercado Pago — somente dados necessarios a transacao
  • Autoridades competentes: quando exigido por lei ou ordem judicial

Nao vendemos, alugamos ou compartilhamos seus dados para fins publicitarios.

6. Retencao dos Dados

Dados sao mantidos enquanto a conta estiver ativa. Apos solicitacao de exclusao pelo titular, os dados sao removidos em ate 30 dias, exceto quando houver obrigacao legal de retencao (fiscal, contabil ou judicial).

7. Direitos do Titular

Conforme o Art. 18 da LGPD, voce tem direito a:

  • Confirmacao da existencia de tratamento
  • Acesso aos seus dados
  • Correcao de dados incompletos ou desatualizados
  • Anonimizacao, bloqueio ou eliminacao de dados desnecessarios
  • Portabilidade dos dados
  • Informacao sobre compartilhamento
  • Revogacao do consentimento

Para exercer qualquer direito, entre em contato: diniz@dcgseguros.com.br ou WhatsApp (11) 93075-9163.

8. Medidas de Seguranca

  • Senhas criptografadas com bcrypt
  • Comunicacao via HTTPS/TLS em todo o site
  • Acesso ao banco de dados restrito por credenciais e IP
  • Backups automatizados diarios
  • Monitoramento de acessos e logs de auditoria
  • Tokens JWT com expiracao para autenticacao

9. Cookies

Este site utiliza apenas cookies estritamente necessarios para funcionamento tecnico (autenticacao, sessao). Nao utilizamos cookies de rastreamento publicitario ou analytics de terceiros.

10. Alteracoes nesta Politica

Esta politica pode ser atualizada periodicamente. Alteracoes relevantes serao comunicadas por e-mail ou aviso no site. A data da ultima atualizacao esta indicada no topo deste documento.
